# 🌌 Mientras el mundo dormía

> "Algunas personas llegan a nuestra vida sin hacer ruido... y terminan cambiándola para siempre."

---

## ¿Qué es este proyecto?

**Mientras el mundo dormía** es una experiencia web interactiva creada para una sola persona.

No es una página tradicional.

No es una carta.

No es una presentación.

Es un recorrido emocional construido con HTML, CSS y JavaScript.

---

## Filosofía

Cada línea de código existe por una razón.

Cada animación tiene un significado.

Cada silencio comunica algo.

Nada está puesto únicamente porque se vea bonito.

---

## Objetivo

Transmitir gratitud.

Transmitir paz.

Transmitir amor.

Nunca culpa.

Nunca presión.

Nunca manipulación.
